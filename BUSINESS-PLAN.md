## 📦 DIE 5 MODULE

### Modul 1 — AI Chatbot (Lead-Qualifizierung)
**Was es macht:**
- 24/7 Beantwortung von Anfragen auf der Website
- Lead-Qualifizierung: "Suchen Sie XY? Wann brauchen Sie es? Wie groß ist das Projekt?"
- Direkte Weiterleitung an Terminbuchung oder WhatsApp
- Notfall-Routing (falls dringend)

**Tech-Stack:** Voiceflow / Tidio / Custom GPT-4o Chatbot via API
**Pain gelöst:** Verpasste Anfragen außerhalb der Öffnungszeiten (60-70% aller Leads!)

### Modul 2 — Automatisches Follow-up (WhatsApp + Email)
**Was es macht:**
- Lead antwortet nicht? Automatisch 3-5 Follow-ups über 7 Tage
- WhatsApp: "Hey [Name], haben Sie noch Fragen zu Ihrer Anfrage?" mit personalisierten Texten
- Email: Professionelle HTML-Templates mit Bewertungen, Portfolio, CTA
- Segmentierung: Warm/Kalt/Heiß

**Tech-Stack:** Twilio/WhatsApp Business API + SendGrid/Resend
**Pain gelöst:** 80% der Leads werden nie nachgefasst. Das macht jetzt die KI.

### Modul 3 — AI Terminbuchung
**Was es macht:**
- Interessent kann direkt im Chat einen Termin buchen
- Kalender-Integration (Google Calendar, Outlook, Calendly)
- Automatische Erinnerungen 24h + 2h vorher
- Stornierung & Umbuchung selbstständig

**Tech-Stack:** Cal.com API oder Calendly API
**Pain gelöst:** Telefonterror für Terminvereinbarungen entfällt komplett.

### Modul 4 — Reputations-Automatisierung (Google Reviews)
**Was es macht:**
- Nach jedem Auftrag: automatische Review-Anfrage per WhatsApp/SMS/Email
- "War alles gut? Wir freuen uns über eine Google-Bewertung" + direkter Link
- Negative Erfahrungen: intern weiterleiten statt öffentlich
- Monatlicher Review-Report

**Tech-Stack:** Google Business Profile API + WhatsApp/SMS
**Pain gelöst:** Gute Kunden bewerten nie. Schlechte Kunden immer. Das drehen wir um.

### Modul 5 — Lead Scraping + Erstes Outreach
**Was es macht:**
- Monatlich X neue qualifizierte Leads in der Region scrapen
- Automatisiertes erstes Outreach (Email + LinkedIn wenn vorhanden)
- CRM-Integration: Leads direkt im Dashboard sichtbar
- Optional: Übergabe an Cold Caller für Telefon-Follow-up

**Tech-Stack:** Apollo.io API / Eigenes Scraping Tool (iSale.deals Synergy!)
**Pain gelöst:** Kein passiver Lead-Eingang. Wir bringen aktiv neue potenzielle Kunden.

## 🎯 ZIELGRUPPE — DEEP ANALYSIS

### Top-Tier (höchste Conversion, bereit zu zahlen):

**1. Zahnarztpraxen & Kieferorthopäden** ⭐⭐⭐⭐⭐
- Durchschnittsumsatz: 400k-1M€/Jahr
- Pain: Telefon klingelt nonstop, keine Rezeption nachts, Neupatienten verloren
- ROI-Story: 1 neuer Zahlungspatient = 2.000-8.000€ Lifetime Value → 1 Monat bezahlt sich selbst
- Entscheider: Praxisinhaber (direkt erreichbar!)
- Conversion-Rate Cold Call: HOCH (konkret ROI erklärbar)

**2. Physiotherapie & Wellness-Studios** ⭐⭐⭐⭐⭐
- Durchschnittsumsatz: 150k-500k€/Jahr
- Pain: Terminmanagement-Chaos, No-Shows kosten Geld
- ROI-Story: Terminbuchung automatisiert + 30% weniger No-Shows = direkte Ersparnis
- Besonderheit: WhatsApp wird bereits aktiv genutzt

**3. Schönheits-Studios (Kosmetik, Nail, Lashes, PMU)** ⭐⭐⭐⭐⭐
- Durchschnittsumsatz: 80k-250k€/Jahr
- Pain: Terminbuchung per Instagram DM, Review-Aufbau fehlt
- ROI-Story: Review-Automation + automatische Wiederbuchung = 20-30% mehr Stammkunden
- Besonderheit: Sehr tech-affin, Instagram-aktiv, jung

**4. Handwerksbetriebe (SHK, Elektriker, Maler)** ⭐⭐⭐⭐
- Durchschnittsumsatz: 500k-3M€/Jahr (Betriebe mit 5-20 MA)
- Pain: Angebote werden nicht nachgefasst, Kunden melden sich nicht zurück
- ROI-Story: Follow-up Automation → 1 gewonnener Auftrag deckt 6 Monate Abo
- Achtung: Entscheider schwerer zu erreichen, oft Skepsis gegenüber Tech

**5. Rechtsanwälte & Steuerberater** ⭐⭐⭐⭐
- Durchschnittsumsatz: 200k-800k€/Jahr
- Pain: Erstanfragen gehen verloren, Terminvereinbarung aufwendig
- ROI-Story: 1 gewonnener Mandant = 5.000-50.000€
- Achtung: Konservative Branche, Vertrauen braucht Zeit

**6. Kliniken & Facharztpraxen** ⭐⭐⭐
- Hohes Budget, aber: DSGVO-Sensibilität, längerer Sales-Cycle

### Zielgruppen-Priorität für Launch:
**Phase 1:** Zahnarzt + Physio + Beauty (schnellste Conversion, konkretester ROI)
**Phase 2:** Handwerk + Rechtsanwälte
**Phase 3:** Kliniken (komplexer, aber höchster LTV)

## 📞 SALES-STRATEGIE

### Kanal 1 — Cold Calling (Haupt-Kanal Launch)

**Team-Aufbau:**
- 2 Cold Caller mit 500€ Provision pro Abschluss
- 1 Closer (du oder Kostas) für Demo-Calls
- Ziel: 5-10 Qualifizierungen/Tag pro Caller → 2-3 Demo-Calls/Woche → 1-2 Abschlüsse/Monat/Caller

**Cold Call Script — Einstieg:**
```
"Guten Tag, ich spreche mit [Name]? Super. Mein Name ist [Name], ich rufe kurz an 
wegen Ihrer Online-Präsenz. Ich frage mich — verlieren Sie eigentlich Kunden, 
weil niemand außerhalb der Öffnungszeiten auf Anfragen antwortet? 
[Pause — sie sagen JA] 
Genau das haben wir für [Branche] in [Stadt] gelöst. Darf ich Ihnen in 
15 Minuten zeigen wie?"
```

**Einwand-Handling:**
- "Kein Interesse" → "Darf ich fragen, haben Sie das Problem X aktuell gelöst?"
- "Zu teuer" → "Was kostet Sie ein verlorener Neukunde im Monat?"
- "Ich mach das selbst" → "Perfekt. Dann schauen Sie, ob unser System das besser macht."
- "Schicken Sie Info zu" → "Ich schick Ihnen gerne was. Wann haben Sie 5 Min für einen kurzen Call?"

### Kanal 2 — Email-Kampagne (13.000 Leads)

**Segmentierung der 13k Liste:**
1. Zahnarztpraxen → Custom Email mit Review-Fokus
2. Physiotherapie → Custom Email mit Termin-Automatisierung
3. Beauty/Wellness → Custom Email mit Wiederbuchungs-Story
4. Handwerk → Custom Email mit Angebots-Follow-up Story
5. Rest → Generische Variante

**Email-Sequenz (5-teilig):**
1. **Tag 1** — Aufmacher: "Wie viele Kunden verlieren Sie täglich nach 18 Uhr?"
2. **Tag 3** — Case Study: "Wie [Branche] in [Stadt] 3 neue Kunden/Monat mit AI gewann"
3. **Tag 7** — Demo-Einladung: "15-Minuten Demo — sehen Sie was AI für Sie tut"
4. **Tag 14** — Urgency: "Noch 3 Plätze in [Region] — dann nehmen wir Mitbewerber"
5. **Tag 21** — Last Call: "Letzte Chance oder ich empfehle jemand anderen"

### Kanal 3 — Content Marketing (Langfristig)

- LinkedIn: Case Studies, ROI-Stories, "AI für [Branche]" Posts
- TikTok/Instagram: "Was passiert wenn ein Handwerker AI nutzt" Videos
- YouTube: "AI Growth System Setup Tutorial" → Vertrauen aufbauen
